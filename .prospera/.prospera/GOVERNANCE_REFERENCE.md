Governance Reference Notice

This repository operates under Prospera OS Governance.

Governance rules, authority models, enforcement logic, and escalation mechanisms are NOT defined in this repository.

This repository implements work that is governed externally.

Authoritative Governance Source

The authoritative governance specifications are maintained in the Prospera OS Governance repository:

ai-governed-software-engineering

All governance interpretation, enforcement, and audit decisions must reference that repository.

Local Repository Limitations

This repository must not be used to:

Define governance rules

Interpret governance authority

Resolve governance conflicts

Override AI participation boundaries

Any governance-related ambiguity must be escalated according to the adopted governance framework.

Human Responsibility

Human maintainers remain fully responsible for understanding and complying with governance requirements.

AI systems do not interpret governance.

AI systems do not resolve governance ambiguity.
